QUE:5 What are undeclared and undefined variables ?



UNDECLARED VARIABLE:

Undeclared variables are those that have not been declared or defined in the current scope..


EXAMPLE

console.log(x); // Error: x is not defined
-> in this example x is not defined in the current scope....


UNDEFINE VARIABLE:

When a variable is declared without value it is called undefined veriable.


EXAMPLE

let y; ,
console.log(y); // undefined
